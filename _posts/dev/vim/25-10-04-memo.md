## marks
- m으로 마크하고, delm + key 로 마크 삭제
- delm! 전부 삭제
- ` 누르면 마크 목록

## keywordprg
- <shift> + k 하면 키워드 설명
- 아래와 같이 man 페이지로 설정 가능
```vim
:set keywordprg=man
```

## unodhistory
- u 로 undo, <C-r> 로 redo
- :undolist 로 undo 히스토리 확인
- :undotree 로 undo tree 확인 (undotree 플러그인 설치 필요)
