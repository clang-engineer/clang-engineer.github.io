## ✅ 왜 생산성 앱을 쓰는가?

* 반복 작업 최소화
* 키보드 중심 작업 속도 향상
* 창(윈도우) 관리 효율 개선
* macOS 기본 기능의 한계 보완

---

## ⭐ macOS 대표 생산성 앱 3대장

### 1️⃣ Raycast

**Spotlight 대체 + 생산성 허브**

* 앱/파일/명령 빠른 검색
* 클립보드 히스토리
* 스니펫, 계산기, 번역
* 윈도우 관리 기능 포함
* 수천 개의 확장(extension) 지원

**요약**

* 현재 macOS 생산성 앱 중 가장 인기
* 무료로도 충분히 강력
* AI, 클라우드 동기화는 유료(Pro)

---

### 2️⃣ Rectangle

**윈도우(Window) 관리 표준 앱**

* 화면 좌/우/상/하 분할
* 키보드 단축키 기반 창 이동
* 마우스 드래그 스냅 지원
* 가볍고 안정적

**버전**

* Rectangle: 무료
* Rectangle Pro: 유료 (영구 라이선스)

**요약**

* 창 정리에 특화된 최고의 선택
* Raycast와 함께 사용되는 경우가 많음

---

### 3️⃣ BetterTouchTool (BTT)

**입력 장치 커스터마이즈 & 자동화 도구**

* 트랙패드 / 마우스 / 키보드 제스처 설정
* 앱별 단축키 및 제스처 지정
* 스크립트 실행, 워크플로 자동화
* Touch Bar 커스터마이즈

**라이선스**

* 45일 무료 체험
* 유료 (2년 / 평생 라이선스)

**요약**

* 고급 사용자 필수 앱
* 자동화와 커스터마이즈의 끝판왕

---

## 🔍 Raycast vs Rectangle vs BTT 간단 비교

| 구분    | Raycast   | Rectangle | BetterTouchTool |
| ----- | --------- | --------- | --------------- |
| 주요 역할 | 런처/생산성 허브 | 창 관리      | 입력/자동화          |
| 난이도   | 낮음        | 매우 낮음     | 높음              |
| 무료 사용 | 가능        | 가능        | 체험만 가능          |
| 추천 대상 | 모든 사용자    | 창 정리 위주   | 파워 유저           |

---

## 🔥 많이 쓰이는 조합

* **Raycast + Rectangle**

  * 가장 대중적인 조합
  * 생산성 + 창 관리 균형

* **Raycast + BetterTouchTool**

  * 키보드/제스처 중심 작업

* **Raycast + Rectangle + BTT**

  * 생산성 풀 세트 (개발자/파워 유저)

---

## 🧩 기타 유명한 macOS 유틸 앱

* **Alfred**: 전통적인 런처 앱
* **AltTab**: Windows 스타일 창 전환
* **CleanShot X**: 스크린샷 끝판왕
* **Keyboard Maestro**: 매크로 자동화
* **Maccy**: 가벼운 클립보드 매니저
* **Homebrew**: macOS 패키지 관리자

---

## ✨ 마무리

macOS 생산성 앱은 "많이 설치하는 것"보다 **본인 작업 스타일에 맞게 선택하는 것**이 중요합니다.

* 입문자: **Raycast + Rectangle**
* 고급 사용자: **Raycast + BTT (+ Rectangle)**

필요한 만큼만 추가해도 작업 효율은 확실히 올라갑니다.
