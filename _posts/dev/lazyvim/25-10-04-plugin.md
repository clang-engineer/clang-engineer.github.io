## 🧠 AI / Copilot 관련

### **copilot.lua**
GitHub Copilot 기본 플러그인. 자동 코드 제안 기능을 제공함.  
`<C-j>` 또는 `<Tab>`으로 제안을 수락할 수 있음.

### **CopilotChat.nvim**
Copilot과 채팅 인터페이스를 연결하는 플러그인.  
Neovim 내에서 AI에게 직접 질문하거나 코드 리뷰 요청 가능.

### **blink-copilot** / **blink.cmp**
Copilot을 `nvim-cmp`와 통합해 자동 완성창에 제안 표시.  
코드 자동완성 UX를 개선함.

### **claude-code.nvim**
Anthropic Claude API 기반의 AI 코딩 어시스턴트 플러그인.  
파일 단위 분석, 코드 요약, 리팩토링 요청 등 가능.

### **codex.nvim**
OpenAI Codex 계열 AI 모델을 활용한 코드 보조 플러그인.  
입력 프롬프트 기반 코드 생성에 사용됨.

## 🎨 UI / 디자인 관련

### **bufferline.nvim**
버퍼(파일)를 탭처럼 표시해주는 상단 바.  
- `<leader>bp`: 현재 버퍼 핀(Pin)  
- `<leader>bP`: 핀 제외 버퍼 닫기  
- `<leader>bd`: 현재 버퍼 닫기  

### **lualine.nvim**
하단 상태줄(Status line) 플러그인.  
파일명, branch, LSP 상태 등을 표시.

### **catppuccin**, **tokyonight.nvim**, **gruvbox.nvim**
대표적인 컬러 테마 플러그인.  
`catppuccin`은 LazyVim 기본 테마로 자주 사용됨.

### **noice.nvim**
명령줄(`:`), 메시지, LSP 알림을 예쁘게 표시하는 UI 개선 플러그인.

### **which-key.nvim**
리더키(`leader`) 누를 때 가능한 키맵 목록을 보여주는 헬퍼.  
LazyVim의 키맵 학습에 필수.

### **render-markdown.nvim**
Markdown 미리보기 및 시각적 렌더링 플러그인.

### **markdown-preview.nvim**
브라우저 기반 실시간 Markdown 미리보기 (`:MarkdownPreview`).

## 🧩 편의성 / 에디팅 관련

### **mini.ai**
문장, 블록, 따옴표 등 “텍스트 객체” 조작 개선 플러그인.

### **mini.comment**
`gc` 단축키로 빠른 주석 토글 기능.

### **mini.pairs**
자동 괄호 닫기 기능 (`(` → `()`).

### **mini.surround**
`ys`, `ds`, `cs`로 괄호, 따옴표, HTML 태그 등 감싸기 조작.

### **flash.nvim**
`/` 없이 즉석에서 단어 검색 & 점프 기능.  
`<leader>s`로 트리거.

### **todo-comments.nvim**
`TODO`, `FIXME`, `NOTE` 등의 주석을 하이라이트하고 모아보기 (`:TodoTelescope`).

### **ts-comments.nvim**
Treesitter 기반으로 더 정확한 주석 토글 기능 제공.

## ⚙️ 개발 / LSP / 포맷팅

### **nvim-lspconfig**
LSP(Language Server Protocol) 설정을 쉽게 도와줌.  
자동완성, 진단, 코드 액션 등의 기반 플러그인.

### **mason.nvim**, **mason-lspconfig.nvim**, **mason-nvim-dap.nvim**
LSP 서버, DAP(Debug Adapter Protocol), 포매터 설치 관리 플러그인.

### **conform.nvim**
포매터(Formatter) 자동 실행 — 저장 시 코드 포맷팅 담당.

### **nvim-lint**
코드 린터 실행 및 경고 표시.

### **nvim-treesitter**
언어 구문 분석기.  
문법 하이라이팅, 코드 블록 인식, Folding 등 핵심 기능 담당.

### **nvim-treesitter-textobjects**
Treesitter 기반 텍스트 객체 확장 (예: 함수, 블록 선택).

### **nvim-ts-autotag**
HTML/XML 태그 자동 닫기.

### **nvim-ts-context-commentstring**
파일 타입에 맞게 올바른 주석 문자열 적용.

### **nvim-jdtls**
Java 전용 LSP 지원 (Maven/Gradle 프로젝트용).

## 🧰 툴 / 탐색 / 검색

### **grug-far.nvim**
`ripgrep` 기반 전역 검색/치환 툴.  
`:GrugFar` → 프로젝트 전체 검색  
`:GrugFarWithin` → 현재 폴더 내 검색

### **persistence.nvim**
Neovim 세션 자동 저장/복원 (`:RestoreSession`).

### **trouble.nvim**
진단/에러/경고 리스트를 트리 형태로 표시.  
`<leader>xx`로 열기.

### **lazy.nvim**
LazyVim의 플러그인 관리 핵심.  
`:Lazy` 명령으로 UI 실행 가능.

### **lazydev.nvim**
LazyVim 개발 편의성 관련 유틸.

## 🧾 Git 관련

### **lazygit.nvim**
내장 터미널에서 LazyGit UI를 바로 실행 (`:LazyGit`).

### **gitsigns.nvim**
Git 변경사항(+/-/~)을 사이드바에 표시.  
- `]c`, `[c`: 다음/이전 변경  
- `<leader>hp`: diff 미리보기  

---

## 🐞 디버깅 / DAP

### **nvim-dap**
Neovim용 디버거 프로토콜.  
브레이크포인트 설정, 실행 중단, 스텝 단위 실행 등 가능.

### **nvim-dap-ui**
DAP용 GUI 패널 제공.  
변수/콜스택/중단점 시각화.

### **nvim-dap-virtual-text**
디버깅 중 변수값을 코드 옆에 표시.

---

## 💾 DB / 데이터 관련

### **vim-dadbod**
Neovim에서 SQL 실행 가능.  
`:DB` 명령으로 DB 접속.

### **vim-dadbod-ui**
`vim-dadbod`용 인터페이스. 테이블 탐색, 쿼리 실행 가능.

### **vim-dadbod-completion**
SQL 자동완성 기능 추가.

## ⚡ 기타

### **plenary.nvim**
많은 플러그인의 의존성 라이브러리 (Lua 유틸 함수 모음).

### **nui.nvim**
Neovim UI 구성용 Lua 컴포넌트 프레임워크.

### **nvim-nio**
비동기 I/O 지원용 Lua 유틸.

### **nvim-nocut**
비정상적인 yank/delete 시에도 레지스터 보호.

### **snacks.nvim**
LazyVim의 UI 개선 요소 모음 (알림, input box, progress 등).

### **friendly-snippets**
`nvim-cmp`와 함께 사용하는 코드 스니펫 모음집.

## 📚 참고 명령

| 명령 | 설명 |
|------|------|
| `:Lazy` | 플러그인 상태 / 업데이트 UI |
| `:Mason` | LSP / DAP / Formatter 설치 UI |
| `:TroubleToggle` | LSP 문제 목록 표시 |
| `:TodoTelescope` | TODO/FIXME 등 검색 |
| `:GrugFar` | 전역 검색 / 치환 |
