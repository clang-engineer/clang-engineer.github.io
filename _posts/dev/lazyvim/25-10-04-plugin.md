# ⚡ LazyVim 통합 플러그인 & 단축키 정리

---

## 🖥️ Core / Manager

| 플러그인                    | 단축키 / 명령          | 설명 / 자동 적용           |
| ----------------------- | -------------- | ---------------------- |
| **LazyVim / lazy.nvim** | `:Lazy`        | 플러그인 설치/업데이트/삭제 관리 대시보드 |

|                         | `:Lazy health` | 환경 점검                 |

## ✍️ Coding / Editor

| 플러그인                   | 단축키 / 명령               | 설명 / 자동 적용             |
| ---------------------- | ---------------------- | ------------------------- |
| **mini.comment**       | `gcc`, `gbc`           | 한 줄/블록 주석 토글 / 자동 적용 |
| **mini.pairs**         | 자동 적용                  | 괄호, 따옴표 쌍 자동 삽입        |
| **mini.surround**      | `ysiw"`, `ds"`, `cs"'` | 감싸기, 삭제, 교체            |
| **which-key.nvim**     | `<leader>` 누르면 팝업      | 키맵 미리보기                   |
| **gitsigns.nvim**      | `]c` / `[c`            | 변경된 hunk 간 이동             |
|                        | `<leader>gp`           | 변경 미리보기                   |
|                        | `<leader>gb`           | blame 보기                     |
|                        | `<leader>gr`           | 변경 리셋                       |
| **flash.nvim**         | `s` / `S`              | 단어/문장 빠른 점프             |
|                        | `r` / `R` (Visual 모드)  | 교체 모드에서 점프              |
| **bufferline.nvim**    | `:BufferLinePick`      | 버퍼 탭 선택                   |
|                        | `<leader>bp`           | 이전 버퍼                      |
|                        | `<leader>bn`           | 다음 버퍼                      |
|                        | `<leader>bd`           | 현재 버퍼 닫기                 |
| **grug-far.nvim**      | `<leader>sr`           | 프로젝트 전체 검색/치환 UI      |
| **vim-visual-multi**   | `<C-n>` / `<C-Down>`   | 멀티 커서 추가 / 여러 위치 동시 편집 |
| **lazygit.nvim**       | `<leader>gg`           | LazyGit UI 실행                 |

# 💡 실무 팁 / 자주 쓰는 조합

| 작업                     | 추천 단축키 / 플러그인 조합                     |
| ---------------------- | ---------------------------------------- |
| AI 코드 완성 / 추천      | 자동 적용 (copilot.lua + blink-copilot) |
| 코드 질문 / 리팩토링      | `<leader>aa` / `<leader>ap` (CopilotChat) |
| 코드 블록 주석            | `gcc` / `gbc` (mini.comment)             |
| 문자열/괄호 감싸기         | `ysiw"`, `ds"`, `cs"'` (mini.surround)  |
| 디버깅 / 변수 확인         | `<F5>`, `<leader>db`, `<leader>dr` (DAP)|
| 함수 이름 변경            | `<leader>rn` (LSP rename)                |
| Git 변경 확인             | `<leader>gp` + `<leader>gr`             |
| 프로젝트 전체 검색/치환    | `<leader>sr` (grug-far.nvim)            |
| Markdown 편집 / 미리보기    | lang.markdown                            |
| SQL / DB 작업             | lang.sql + vim-dadbod 자동완성          |
| TypeScript / JS 작업      | lang.typescript                           |
