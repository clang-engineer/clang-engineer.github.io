# 장구피의 특징
- 소백피	대중적이며 주로 사물, 반주, 무용에 사용
- 소가죽	대중적이며 주로 사물, 풍물, 반주에 사용
- 개가죽	두툼하고 부드러우며 소리가 높다. 양피에 비해 내구도가 좋아 주로사물놀이용으로 사용
- 양가죽	장구피중 가장 부드럽고 맑은 소리를 내며, 내구도가 약해 풍물굿, 민요, 가락등의 용도로 사용
- 말가죽	전문가들이 사용하는 고급소재로 주로 사물, 풍물에 사용
